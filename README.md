# Functional Storage Lgeacy

Functional Storage Lgeacy 是一个将高版本 Functional Storage 玩法回移到 Minecraft 1.12.2 的模组。

- Mod ID: functionalstoragelgeacy
- Minecraft: 1.12.2
- Java: 8
- License: MIT

## 功能概览

### 抽屉与存储
- 木质抽屉：1 格、2 格、4 格
- 特殊抽屉：压缩抽屉、简易压缩抽屉、末影抽屉、流体抽屉（1/2/4 槽）
- 盔甲柜（Armory Cabinet）

### 控制器网络
- 存储控制器与控制器拓展
- 使用 Linking Tool 进行单个/批量连接与移除

### 升级系统
- 存储升级：铁降级、铜、金、钻石、下界合金、创造售卖
- 功能升级：虚空、红石、拉取、推送、收集
- 升级计算逻辑已与高版本方向对齐（乘法倍率体系）

### 交互与渲染
- 抽屉正面内容渲染、数量显示、升级图标显示
- 流体抽屉支持使用桶/流体容器进行存取
- 配置工具支持锁定、渲染开关、数字显示、指示条模式

## 主要交互说明

### Linking Tool
- 右键控制器：记录目标控制器
- 右键抽屉：执行连接或移除
- 潜行右键空气：切换模式（单个/批量、添加/移除）

### Configuration Tool
- 潜行右键空气：切换配置模式
- 右键抽屉：应用当前配置（锁定、数字显示、渲染、升级显示、指示条）
